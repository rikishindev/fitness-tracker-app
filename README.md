# fitness-tracker-app
Fitness Task Tracker App (Todo + Timer)
## 概要
本アプリは、日々の筋トレ習慣をもとに作成しました。
Todoリスト機能とタイマー機能を組み合わせ、日々のトレーニングを効率的に管理できます。
---

## 機能

- タスクの追加・削除・編集
- タスクの完了状態管理
- タイマー機能（開始・一時停止・リセット）
- 記録時間の保存（localStorage）

---

## 使用技術

- HTML
- CSS
- JavaScript（DOM操作）
- localStorage

---

## 工夫した点

- データとUIを分離し、状態に応じて再描画する設計にしました  
- イベントデリゲーションを使用し、効率的なイベント管理を実装しました  
- localStorageを使用して、ページをリロードしてもデータが保持されるようにしました
